#
# maps 8 notes to the C Major triads
#

OnNote Any NoteC1 Any
	Print "I  : C Major"
	SendNote Channel NoteC3 Velocity
	SendNote Channel NoteE3 Velocity
	SendNote Channel NoteG3 Velocity
End

OnNote Any NoteDb1 Any
	Print "ii : D Minor"
	SendNote Channel NoteD3 Velocity
	SendNote Channel NoteF3 Velocity
	SendNote Channel NoteA3 Velocity
End

OnNote Any NoteD1 Any
	Print "iii: E Minor"
	SendNote Channel NoteB2 Velocity
	SendNote Channel NoteE3 Velocity
	SendNote Channel NoteG3 Velocity
End

OnNote Any NoteEb1 Any
	Print "iv : F Minor"
	SendNote Channel NoteC3 Velocity
	SendNote Channel NoteF3 Velocity
	SendNote Channel NoteAb3 Velocity
End

OnNote Any NoteE1 Any
	Print "IV : F Major"
	SendNote Channel NoteC3 Velocity
	SendNote Channel NoteF3 Velocity
	SendNote Channel NoteA3 Velocity
End

OnNote Any NoteF1 Any
	Print "V  : G Major"
	SendNote Channel NoteB2 Velocity
	SendNote Channel NoteD3 Velocity
	SendNote Channel NoteG3 Velocity
End

OnNote Any NoteGb1 Any
	Print "vi : A Minor"
	SendNote Channel NoteC3 Velocity
	SendNote Channel NoteE3 Velocity
	SendNote Channel NoteA3 Velocity
End

OnNote Any NoteG1 Any
	Print "V7 : G Major 7th"
	SendNote Channel NoteB2 Velocity
	SendNote Channel NoteD3 Velocity
	SendNote Channel NoteF3 Velocity
	SendNote Channel NoteG3 Velocity
End

OnElse
	SendCopy
End
